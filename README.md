# Equipping Pastors Worldwide

Marketing + devotions site for **Equipping Pastors Worldwide (EPW)** — a missions
nonprofit training pastors on the frontlines of the Global South.

- **Stack:** Vite + React 19 + Tailwind 4, static prerender (SSG) → Netlify
- **Build:** `npm run build` (tsc + vite client + vite SSR + prerender → `dist/`)
- **Dev:** `npm run dev` (port 5219)
- **Deploy:** push to `main` → Netlify auto-build & deploy
- **Domain:** epwusa.org · giving processed at epworldwide.org/support/donate-funds
- **Add a devotion:** append an entry to `src/data/devotions.ts`

Site by Adam Loomis Marketing.
